Tujuan dari praktikum ini adalah:

1. Memahami konsep dasar framework dalam pengembangan aplikasi web.

## Kesimpulan

Dari praktikum ini dapat disimpulkan bahwa:

1. Framework CodeIgniter mempermudah pengembangan aplikasi web.

2. Konsep MVC membantu memisahkan logika program, data, dan tampilan.

3. Routing digunakan untuk mengarahkan request ke controller yang sesuai.

4. Layout template dapat digunakan untuk membuat tampilan web lebih terstruktur.
